# ✈️ SkyWings — Premium Travel & Flight Booking Platform

[![Tailwind CSS 4](https://img.shields.io/badge/Tailwind-v4-06B6D4?style=flat-square&logo=tailwindcss)](https://tailwindcss.com/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-12.x-E10098?style=flat-square&logo=framer)](https://www.framer.com/motion/)

**SkyWings** is a production-grade, full-service flight ticket booking and travel agency web application built with Next.js 16 (App Router), TypeScript, and Tailwind CSS. Designed with an Emirates & Qatar Airways inspired luxury travel aesthetic, it features an interactive 283-frame cinematic scroll sequence, flight search widget, popular destinations carousel, and multi-service travel offerings.

---

## ✨ Features

- **🎬 Cinematic Intro Scroll Animation**: 283-frame high-DPI HTML5 canvas sequence scrubbing with Framer Motion spring physics.
- **🛫 Flight Search Engine**: Real-time flight search widget with trip type selection (Round Trip, One Way, Multi-City), airport swap animation, traveller counter, and cabin class selection.
- **🎨 Premium Design System**: Custom HSL color tokens, typography hierarchy (Playfair Display, Inter, JetBrains Mono), glassmorphism effects, and smooth scroll reveals.
- **🌍 Complete Travel Services**: Flight bookings, hotel reservations, visa processing, tour packages, travel insurance, and corporate travel solutions.
- **📱 Fully Responsive**: Optimized mobile navigation drawer, fluid typography, and adaptive layouts across all viewports.
- **⚡ Production Ready**: SEO metadata, OpenGraph cards, optimized next/font loading, and zero build warnings.

---

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router & Turbopack)
- **Language**: TypeScript
- **Styling**: Vanilla CSS tokens & Tailwind CSS 4
- **Animations**: Framer Motion 12
- **Icons**: Lucide React
- **Form Management**: React Hook Form & Zod

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jamilgit19/skywings-travel-platform.git
   cd skywings-travel-platform
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env.local
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
